# SHERLOOK
**Super Hybrid Efficient Reasoning Layered Orchestrator Of Knowledge**

Motor de IA híbrido con:
- Parser de reglas tipo DCG-lite
- Solvers simbólicos (SymPy, Z3)
- Pronósticos optimizados (PuLP)
- Clasificación híbrida (MLP + DecisionTree)
- Blackboards colaborativos
